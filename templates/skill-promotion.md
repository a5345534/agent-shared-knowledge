# Promote to Reusable Skill

Use when an inbox candidate (or curated workspace memory entry) describes a repeatable agent procedure, command workflow, template, or script-backed capability.

Destination: `agent-workspace/skills/<skill>/`

Rules:

- This is review-gated; hooks do not auto-apply skill promotions.
- Skills live under `agent-workspace/skills/` when they support development agents across sessions.
- Keep a concise pointer in shared memory or mark the original entry `type: deprecated`.
